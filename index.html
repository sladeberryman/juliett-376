User Interface

This site showcases an example of a user interface that could benefit from improvements, along with a proposed solution.

Example: UNA Parking Portal Vehicle Activation

  • The UNA Parking Portal's vehicle activation functionality requires too many clicks for students who are in a hurry.
  • On this site, I present a shortened version of that workflow that has a card that would be in each student's UNA Portal Page.
  • I know it isn't an exact replica of Portal or a beautiful interface by any means, but I just wanted a way to change the Active Vehicle directly in Portal
  • Should just be a couple API calls to implement this solution in Portal
