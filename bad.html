Step 1

The first step for a user is logging into their UNA portal.

Step 2

To access their activated vehicles, students must first get into their parking portal, which can be accessed by scrolling down to the "Personal Information" card.

Step 3

User is then directed to the Parking Portal login page

Step 4

Upon Login, user is brought to Parking Portal home screen. User must then click "Vehicles" to edit vehicles.

Step 5

Finally, the user must use the Active/Inactive selection button to toggle which vehicle they want to be active.
